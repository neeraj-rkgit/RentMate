<b> RentMate <b>
RentMate is a web-based platform that connects landlords and tenants. Landlords can list their properties with details like price, location, and amenities, while tenants can search for and inquire about available rentals. Key features include property listings, search filters, user accounts, messaging, and an admin dashboard. The platform streamlines the rental process, making it easier for both parties to communicate and manage listings.
<br>
<br>
Tools Used: <br>
Frontend: React.js, Tailwind CSS, Bootstrap <br>
Backend: Node.js, Express.js <br>
Database: MongoDB or MySQL <br>
